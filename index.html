Premi

Premi is a web application with whom the user can create and edit slideshows as happens with Prezi, Strut, Slides and others.

It is a didactic project of Software Engineering course held in Università degli Studi di Padova during Academic Year 2014-2015 proposed by Zucchetti S.p.a.

Project specifications are available here (PDF in italian).

What it is made by

Premi uses several frameworks and libraries from the web. Here are some.

  • MEAN.JS

    A full-stack JavaScript open-source solution, which provides a solid starting point for MongoDB, Node.js, Express, and AngularJS based applications.

  • Fabric.js

    A powerful and simple JavaScript HTML5 canvas library.

  • Materialize

    A modern responsive front-end framework based on Material Design.

  • Impress.js

    A presentation framework based on the power of CSS3 transforms and transitions in modern browsers and inspired by the idea behind Prezi.

  • jsPDF

    A HTML5 client-side solution for generating PDFs.
